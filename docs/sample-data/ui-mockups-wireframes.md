    ## 5. レスポンシブフォームのブレークポイント
    
    ### デスクトップ (1200px+)
    ```
    ┌─────────────────────────────────────────────────────────────┐
    │  [フィールド1]  [フィールド2]  [フィールド3]  [フィールド4]   │  横4列
    └─────────────────────────────────────────────────────────────┘
    ```
    
    ### タブレット (768px - 1199px)
    ```
    ┌─────────────────────────────────────┐
    │  [フィールド1]    [フィールド2]     │  横2列
    │  [フィールド3]    [フィールド4]     │
    └─────────────────────────────────────┘
    ```
    
    ### スマートフォン (768px未満)
    ```
    ┌─────────────────┐
    │  [フィールド1]  │  縦1列
    │  [フィールド2]  │
    │  [フィールド3]  │
    │  [フィールド4]  │
    └─────────────────┘
    ```
    
    これらの画像やワイヤーフレームをAIプロンプトと一緒に提供することで、視覚的な情報からもより適切なInput クラス群を生成できるようになります。
